Monthly Blogs

Fast And Smart

MBD AI is powered by posable by Open Source LLMs. A LLM or large langue model is the AI model you the user is using. For Example, if you need help with your code, you chose codegemma because it is designed to help users with code and how to explain it. MBD AI only has 2 LLMs publicly available, llama 3 and codegemma.

llama 3

LLaMA-3 is a large language model developed by Meta AI, capable of generating human-like text based on input prompts. It has been trained on a massive dataset of internet text and can understand and generate text in multiple languages. The model can be fine-tuned for specific tasks such as text classification, sentiment analysis, language translation, and chatbot applications.
- MBD AI llama 3

Codegemma

CodeGemma is an AI-powered coding companion developed by Meta AI that helps software developers with tasks such as code completion, generation, refactoring, bug detection, and code review. The model uses a transformer-based architecture and was trained on a massive dataset of open-source code from various programming languages. By assisting with these tasks, CodeGemma aims to improve the efficiency and accuracy of software development workflows.
